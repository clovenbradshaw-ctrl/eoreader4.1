Two cursors on a corpus about a library and crime

A causal effect is a counterfactual — it is not in any text. So this never says “the library cut crime.” It says what the reader reads the sources as claiming, traced to the passage, typed by the stance the source proposed — and it can never upgrade that stance, because only a design can.

The causal graph the corpus is read as asserting. Each edge is a claim, not a fact; click one to see the passages that made it. Nodes are the described constructs; an amber node is warm from both trails — a candidate common cause the corpus cannot rule out.

Select an edge or a node to trace its claims to their passages.
